
# jSignal
A simple signal based event handler for JavaScript.

## Install

```
npm install jsignal
```

### Using es2015
```javascript
import jSignal from 'jsignal'
import * as jSignal from 'jsignal'
import { default as jSignal } from 'jsignal'
```

### Using CommonJS
```javascript
var jSignal = require('jsignal');
```
